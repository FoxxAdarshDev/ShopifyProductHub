    <h2>VersaCap® 38-430 with 3× 1/8" Hose Barb Ports – Tamper-Evident Closure for Sterile Fluid Management</h2>
    <p>The 38-430 VersaCap® with Tamper Evident Ring is a secure, cleanroom-compatible closure designed for use in bioprocess, pharmaceutical, and laboratory applications. It fits standard 38-430 threaded bottles and provides an added layer of protection for critical fluids and materials by clearly indicating first-time opening.</p>
    <p>This patented cap features a break-away tamper evident ring for irreversible security, along with a molded, non-machined adapter that ensures a clean, reliable seal in sterile environments. The rotating adapter design minimizes tubing twist during handling, while interchangeable polypropylene inserts allow for 2 and 3 port configurations in both 1/8" and 1/4" hose barb sizes. Made from autoclavable polypropylene, the cap is chemical-resistant and built for repeated use.</p>
    <p>Foxx Life Sciences offers custom bottle assemblies with VersaCap® Technology tailored to your unique fluid handling needs—whether for sampling, filling, or storage. Caps can be configured with filters, dip tubes, or connectors to suit a variety of bioprocess workflows. For custom solutions or additional information, please contact our sales team.</p>

    function haloSwitchTab(event, tabId) {
      // Remove active class from all tab buttons
      var tabButtons = document.querySelectorAll('.halo-tab-button');
      tabButtons.forEach(function(button) {
        button.classList.remove('active');
      });

      // Add active class to clicked button
      event.target.classList.add('active');

      // Hide all tab content
      var tabContents = document.querySelectorAll('.halo-tab-content');
      tabContents.forEach(function(content) {
        content.classList.remove('active');
      });

      // Show selected tab content
      var activeTab = document.getElementById('halo-' + tabId);
      if (activeTab) {
        activeTab.classList.add('active');
      }
    }

    // Initialize tabs on page load
    document.addEventListener('DOMContentLoaded', function() {
      // Ensure first tab is active by default
      var firstButton = document.querySelector('.halo-tab-button');
      var firstContent = document.querySelector('.halo-tab-content');

      if (firstButton && firstContent) {
        firstButton.classList.add('active');
        firstContent.classList.add('active');
      }
    });
    </script>
